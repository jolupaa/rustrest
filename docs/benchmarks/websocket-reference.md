# WebSocket Reference Profile

The accepted production reference is generated by:

```bash
./scripts/run-websocket-reference-profile.sh
```

The gate requires Linux, at least 65,536 file descriptors, 10,000 idle
connections, 1,000 active connections, 900 seconds of load, complete echo
delivery, zero unexpected closes, and graceful shutdown within 15 seconds.
Artifacts are written under `target/ws-reference/`.

No successful reference baseline is committed yet. The current development
environment is macOS and cannot produce the required Linux `/proc` and
`/usr/bin/time -v` measurements. Run the script on the designated Linux
reference host and replace this notice only with measured values from a passing
report: date, commit, kernel, CPU, RAM, Rust version, limits, configuration,
connection counts, throughput, p50/p95/p99 latency, peak and stabilized RSS,
CPU, open file descriptors, errors, and shutdown duration.

## Development smoke (not the reference baseline)

On June 12, 2026, a macOS development smoke ran 100 idle and 20 active
connections for 30 seconds with 256-byte messages. It established all 120
connections, sent and received 47,244 exact echoes, and recorded zero connect,
send, receive, or unexpected-close failures. Round-trip latency was 797 us p50,
2,731 us p95, and 7,142 us p99. This result validates the load tool but does not
replace the required Linux 10,000/1,000 reference profile.

A second local scale smoke used the full 10,000 idle and 1,000 active
connection count for 30 seconds on Darwin 25.5.0, Apple M2 Pro (12 logical
CPUs), 16 GiB RAM, Rust 1.96.0, and an open-file limit of 1,048,575. All
11,000 connections opened. The client sent and received 2,375,589 exact
256-byte echoes (about 79,186 messages/second) with zero failures. Latency was
1,301 us p50, 3,239 us p95, and 4,809 us p99. One live sample showed about
166 MiB server RSS, 376 MiB client RSS, 239% server CPU, 247% client CPU, and
22,000 established TCP endpoints. This remains a short macOS smoke rather than
the required 900-second Linux stability result.
